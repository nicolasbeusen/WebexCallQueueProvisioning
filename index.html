How it works
  1. Resolve personId using /v1/people/me
  2. Get locationId using /v1/people/{personId}?callingData=true
  3. List queues via /v1/telephony/config/queues/locationId={locationId}
  4. Fetch assignments via /v1/telephony/config/queues/agents/{personId}?offset&limit
